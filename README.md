# Shale-Namma Pride 🏫✨

## Android App Development using GenAI

Shale-Namma Pride is a **School Transparency & Community Pride Portal** developed for government schools. The app helps parents, students, teachers, volunteers, and school committees stay connected through live updates, school announcements, meals tracking, student achievements, and school information.

The project focuses on improving trust in public education systems by showcasing the daily activities and achievements of government schools.

---

# 📱 Features

## 🔐 Authentication & Role Management

* Firebase Authentication
* Login & Registration System
* Role-based access:

  * Admin
  * Parent
  * Student
  * Volunteer

## 🏫 School Dashboard

* School Information
* Total Students
* Students Present Today
* Teacher Count
* Headmaster Details
* School Address
* Latest Announcements

## 🍲 Daily Meals System

# 🚀 How to Run the Project

## Clone Repository

```bash
git clone https://github.com/n-varsha13/ShaleNammaPride.git
```

---

## Open in Android Studio

Open the project folder in Android Studio.

---

## Connect Firebase

### Add Firebase to Android App

1. Create Firebase Project
2. Add Android App
3. Download `google-services.json`
4. Place file inside:

```plaintext
app/google-services.json
```

---

## Enable Firebase Services

Enable:

* Firebase Authentication
* Realtime Database
* Firebase Storage

---

## Run Application

```plaintext
Shift + F10
```
